# Evan의 기억 공간

> 자동매매 시스템과 엔지니어링에 대해 배운 것들을 기록합니다.

## 소개

AI를 활용한 자동매매 프로그램 개발에 관심을 가지면서 시작한 여정입니다. Oracle 서버부터 한국투자증권 API, Claude까지—다양한 도구와 기술을 경험하고 있습니다. 이 블로그는 그 과정에서 배운 기술적 인사이트와 실무 경험을 기록하는 공간입니다.

---

## 주요 경험

### 자동 거래 시스템
- **kis-auto-trading-bot** | KOSPI 시장 자동 거래 봇
- **upbit-auto-trading-bot** | 암호화폐 거래 자동화

이 프로젝트들을 통해:
- Oracle E2 서버 구성 및 운영
- LLM 활용 (ChatGPT → Gemini → Claude 선정)
- VS Code, GitHub, Claude.md 연동 개발 환경 구축
- 한국투자증권 공식 API 학습 및 실무 적용
